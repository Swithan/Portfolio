Cyber Security Challenge Belgium

Cyber sécurité

Le Cyber Security Challenge est une compétition "Capture the Flag". L'objectif de cet évènement est de résoudre des défis de difficultés variables endéans un temps imparti (32 heures dans ce cas).
J'ai eu l'occasion de participer aux 2 dernières éditions avec d'autres membres de l'Ephec. C'est tout d'abord la présentation sur le sujet qui a été faite à l'Ephec qui m'a donné envie d'essayer cela. Il y avait également le fait d'avoir vu (et aider comme je pouvais) mes amis qui s'étaient inscrits alors que nous n'étions qu'en 1ère.
Ce challenge m'a beaucoup apporté, que ce soit dans la relation avec mes amis, mais aussi dans les compétences techniques que j'ai pu acquérir avec les épreuves.
