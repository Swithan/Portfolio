Formation React JS

Développement Web

Formation réalisée sur openClassrooms. Le but était d'apprendre cette technologie afin de mettre en place mon portfolio. Malheureusement, le temps ne m'a pas permis de concrétiser cela.
React JS est cependant un framework très intéressant, qui s'approche d'Angular (que je connaissais déjà) tout en étant beaucoup plus léger. La formation quand à elle était intéressante et complète. Le code source de la base de mon portfolio est tout de même disponible sur GitHub.
L'apprentissage de react JS n'a finalement pas été d'utilité immédiate, mais la connaissance de ce framework pourrait m'amener à trouver un job dans cette technologie. Le simple fait de faire une formation est également un plus. Apprendre à apprendre reste très important pour beaucoup d'aspect, d'autant plus en programmation car c'est un milieu ou tout évolue extrèmement rapidement.
